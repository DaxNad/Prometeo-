# PROMETEO BOOT PROTOCOL v1.0

## 1. Identità sistema

**PROMETEO** è una piattaforma di orchestrazione e monitoraggio per reparti produttivi.

Funzioni principali:

- raccolta eventi produzione
- monitoraggio stazioni
- dashboard operative
- analisi e supporto decisionale

Architettura API-driven, con evoluzione verso piattaforma completa realtime e persistente.
