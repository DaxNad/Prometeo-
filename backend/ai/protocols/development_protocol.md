# PROMETEO AI DEVELOPMENT PROTOCOL

## Ruoli AI

### ChatGPT
- architettura
- progettazione moduli
- codice completo
- refactor strutturale
- debug guidato

### Perplexity
- documentazione aggiornata
- best practice
- confronto librerie
- pattern reali

### Copilot / IDE AI
- completamento codice
- refactor locale
- test rapidi

## Regole operative
1. prima progettazione
2. poi verifica tecnica
3. poi implementazione
4. ogni decisione importante va registrata in docs/decisions
5. ogni modifica significativa va registrata in board/system_log.md

## Ciclo standard
IDEA
→ MASTER CONTROL
→ TASK BOARD
→ ARCHITECTURE DECISION
→ CODICE
→ SYSTEM LOG
→ TEST
→ DEPLOY

## Regola repository
Durante sviluppo attivo:
- lavorare dal repository locale
- evitare modifiche via GitHub web
- usare GitHub per lettura, cronologia, log e deploy
